Served as a Tutor for machine learning courses, supporting undergraduate students in understanding key concepts and practical applications.

## Courses

### Trimester 1 (March - June 2020)
* **COMP307**: Machine Learning and Data Mining

### Trimester 2 (July - October 2020)
* **COMP309**: Machine Learning Tools and Techniques

## Responsibilities

* Conducted weekly tutorial sessions
* Led hands-on laboratory demonstrations
* Provided individual assistance to students
* Graded assignments and practical exercises
* Held regular office hours for student consultations

## Topics Covered

* Supervised and unsupervised learning algorithms
* Decision trees, neural networks, and ensemble methods
* Feature selection and dimensionality reduction
* Practical implementation using Python and machine learning libraries
* Data preprocessing and model evaluation techniques
